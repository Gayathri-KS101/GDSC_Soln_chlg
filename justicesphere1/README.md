# JusticeSphere Website

A responsive website showcasing the Equit and Stardust applications, their features, and our mission to promote justice and peace through technology.

## Features

- Responsive design
- Smooth scrolling animations
- Social media integration
- Download section for apps
- Modern UI/UX

## Technologies Used

- HTML5
- CSS3
- JavaScript
- AOS (Animate On Scroll) library
- Font Awesome icons
- Google Fonts (Poppins)

## Project Structure

```
justicesphere1/
├── index.html          # Main HTML file
├── style.css          # Stylesheet
├── script.js         # JavaScript code
└── assets/          # Image assets
    └── *.jpg       # Various images
```

## Setup

1. Clone the repository:
```bash
git clone https://github.com/your-username/GDSC_Soln_chlg.git
cd GDSC_Soln_chlg/justicesphere1
```

2. Open `index.html` in a web browser or use a local server:
```bash
python -m http.server 8000
```
Then visit `http://localhost:8000` in your browser.

## Customization

1. Colors and Typography
   - Edit `style.css` for color schemes
   - Modify Google Fonts in `index.html`

2. Content
   - Update text in `index.html`
   - Replace images in corresponding sections

3. Animations
   - Modify AOS attributes in HTML elements
   - Customize animation timing in JavaScript
