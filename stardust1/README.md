# Stardust - Healing Through Guided Yoga

Stardust is a Flutter application designed to support victims of injustice through specially crafted animated yoga sessions and meditation guidance. Our app provides a safe, accessible space for healing and recovery through therapeutic movement and mindfulness practices.

## Features

### Guided Animated Yoga Sessions
- Professionally designed yoga sequences for trauma recovery
- Custom animated video guides for proper form and technique
- Progressive difficulty levels for all abilities
- Specialized routines for stress relief and emotional healing

### Therapeutic Benefits
- Trauma recovery support
- Stress and anxiety reduction
- Physical and emotional healing
- Building resilience and strength
- Mind-body connection enhancement

### User Experience
- Peaceful and calming interface design
- Custom typography for enhanced readability
- Intuitive navigation system
- Progress tracking capabilities
- Offline video support

## Technical Features

- Custom animated yoga instruction videos
- Video player integration for smooth playback
- Intuitive UI with custom fonts (Starstruck, Kinkie-Zpwm)
- Asset management for extensive video content
- Responsive design for various screen sizes

## Prerequisites

Before you begin, ensure you have the following installed:
- Flutter SDK (>=3.2.5)
- Dart SDK
- Android Studio or VS Code with Flutter plugin
- Git

## Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/GDSC_Soln_chlg.git
cd GDSC_Soln_chlg/stardust1
```

2. Install dependencies:
```bash
flutter pub get
```

3. Run the app:
```bash
flutter run
```

## Dependencies

The app uses the following main dependencies:
- `video_player`: For seamless video content playback
- `cupertino_icons`: For iOS style icons

Add them to your project by running:
```bash
flutter pub add video_player
```
